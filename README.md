farth
=====

Forth-like programming language

## Installation ##
<p>
	Run ```python setup.py install```
	<p>
		After that You will be able to run interpreter by running following command:
		```bash
		python -m Farth
		```
	</p>
</p>

## Syntax ##
Syntax is almost the same as in typical Forth language.

### Hello world ###
```forth
"Hello, world!" print
```
